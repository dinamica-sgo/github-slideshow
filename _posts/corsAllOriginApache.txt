Cuando wp bloquea recursos para un dominio CNAME


vim /etc/apache2/sites-enabled/[name of your conf file].conf

<Directory /var/www>
    # some other apache code here, if any
    # replace the url to the one you wanted
    Header set Access-Control-Allow-Origin "*"
    # some other apache code here, if any
</Directory>

/etc/init.d/apache2 restart

sudo a2enmod headers



o sino en el .httacces agregar

Header add Access-Control-Allow-Origin "*"
